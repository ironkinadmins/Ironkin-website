How to Play — Battleship Bingo

Screenshot Proofs

Upload screenshot(s) as evidence for completing a tile or getting a drop. You can drag and drop, paste from clipboard, or click to browse — up to 10 images per submission. Images are automatically compressed before upload.

An admin reviews each proof and approves or rejects it. Check the Proofs tab for all submissions.

Teams

All game types support solo or team play. When joining a team game, you pick your team. Admins can move players between teams and rename teams.

Admin System

The room creator is the default admin. Admins can promote or demote other players, lock or unlock the room, edit tiles, and manage gameplay.

Game Buttons

Generate Board / Import List creates the board from a Google Sheet or pasted item list. Reset resets progress without changing the board. Reroll Board can create new items/positions or shuffle positions. Admin Panel handles tile edits, undo actions, ownership, players, and room settings.

How to Play — Battleship Bingo

A strategic 10×10 bingo board where two teams secretly place classic Battleship ships, then race to get drops and attack the opponent's fleet.

Setup: Staff build the 10×10 board with drops/items, then assign one captain per team. Each captain secretly places six ships on the board: Carrier (5 tiles), Battleship (4), Cruiser (3), Submarine (3), Destroyer (3), and Patrol Boat (2). Ships must be placed in straight lines and cannot overlap.

Gameplay: Once both captains confirm their ship placements, the game begins. Players submit proof of obtaining a drop for any tile on the board. An admin or captain approves the proof, and the tile is automatically attacked on the opponent's board.

Attacks: Each tile attack results in a Hit or Miss. When all cells of a ship are hit, it is sunk and announced to both teams.

Board Views: Toggle between Attack Board and Your Waters to view attacks, misses, hits, and your own fleet.

Winning: Sink all enemy ships for an instant win. If the deadline passes, the team with the most ships sunk wins.

Import Battleship Bingo List

Paste one item per line. Quantities are optional. The importer will fill the 10×10 board from top-left to bottom-right and try to pull item images from the OSRS Wiki.

Examples:

Zenyte shard x3
Abyssal whip x2
Dragon warhammer
Blood shard, 5
Enhanced crystal weapon seed - 1

Admin System

The room creator is the default admin. Optionally, set an admin password when creating the room — anyone who enters it via the 🔑 button gains admin powers. Admins can promote or demote other players from the Admin Panel. The password is stored securely and cannot be seen in browser dev tools.

Game Buttons

Generate Board / Import List — Admin only. Creates the board from a Google Sheet or pasted list of items.

Reset — Admin only. Resets all progress without changing the board.

Reroll Board — Admin only. Choose Full Reroll or Shuffle Positions. Both reset progress.

Admin Panel — Admin only. Edit or replace tiles, undo rolls, move pieces, change territory owners, manage players, lock/unlock the room, and set an admin password.

Teams — Admin only. Rename teams.
